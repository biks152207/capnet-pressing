## 3. Start server

### 3.3 Set up the env variable (Optional)

Add a `.env` in the root folder.

```ruby
NODE_ENV=development
```

### 3.2 NPM

For production and development

```console
npm start
```

### 3.3 Gulp

For development only

```console
gulp
```
